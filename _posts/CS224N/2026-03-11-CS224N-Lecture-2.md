## 1. Optimizations Basics

### **• In The Last Session:**

- **Cost(Loss) Function $$J(\theta)$$**

  $$J(\theta) = -\frac{1}{T} \log L(\theta) = -\frac{1}{T} \sum_{t=1}^{T} \sum_{\substack{-m \le j \le m \\ j \ne 0}} \log P(w_{t+j} \mid w_t ; \theta)$$

- **Probability $$P(w_{t+j} \mid w_t ; \theta)$$**

  $$P(o|c) = \frac{\exp(u_o^T v_c)}{\sum_{w \in V} \exp(u_w^T v_c)}$$   

- **Train the Model**
  - 중심 단어(Center Word: $$v_w$$)가 올 때, 맥락 단어(Context Word: $$u_w$$)가 올 확률을 최대화 하는 것 $$=$$ 손실 함수(Cost Function) $$J(\theta)$$를 최소화 하는 것 
  - 이를 위하여 <mark>경사 하강법(Gradient Decent)를 활용하여 $J(\theta)$의 값을 최소로 만드는 전체 매개변수(parameter) $\theta$의 값을 찾는 것</mark>이 목표  
<br/>

### **• Gradient Descent(경사하강법):**

  ![image](2.png)  

- **Idea**
  - 비용 함수(Cost Function)의 기울기(Gradient)를 계산하여 어떤 방향이 내리막(downhill)인지 파악하여  해당 방향으로 나아가고, 나아간 위치에서 앞의 과정을 반복적으로 수행함  
  - (위의 그림과 같은) 일차원 함수의 경우에는 한 방향의 내리막을 따라가면 되는 단순한 작업이지만, (Word2Vec에서 다루는) 다차원 함수와 같은 경우에는 어떤 지점에서 gradient 계산 시 단방향이 아닌 여러 방향을 따라 내려갈 가능성 존재  
  -> Gradient 계산이 필수적  
  ![image](https://upload.wikimedia.org/wikipedia/commons/d/d2/3d-gradient-cos.svg)_다차원 함수의 형태 예시_  
<br/>

- **Basic Algorithm**
  - **Equation**
    - Update Equation (전체 행렬에 대해서)

      $$\theta^{new} = \theta^{old} - \alpha \nabla_{\theta} J(\theta)$$

    - Update Equation (개별 파라미터 $$\theta_j$$에 대해서)

      $$\theta_j^{new} = \theta_j^{old} - \alpha \frac{\partial}{\partial \theta_j^{old}} J(\theta)$$

    - $$\alpha$$ = Learning Rate or Step Size  
      \: 일반적으로 $$10^{-5}$$ ~ $$10^{-3}$$ 정도 되는 작은 값을 사용  
      \: 너무 큰 $$\alpha$$ 값을 가지게 되면 적절하지 않은 위치(e.g., minima가 아님)로 이동하거나 하는 문제 발생 가능

    > Cf. **편미분(Partial derivative)과 행렬미분(Matrix derivate)**  
    > • [편미분](https://ko.wikipedia.org/wiki/%ED%8E%B8%EB%AF%B8%EB%B6%84)  
    > e.g., $$\frac{\partial}{\partial x} f$$는 함수 $$f(x, y, ...)$$의 편미분을 의미한다  
    > • [행렬미분](https://en.wikipedia.org/wiki/Matrix_calculus)  
    > e.g., $$\nabla f = \left( \frac{\partial f}{\partial \mathbf{x}} \right)^\text{T} = \left[ \frac{\partial f}{\partial x_1} \quad \frac{\partial f}{\partial x_2} \quad \frac{\partial f}{\partial x_3} \right]^\text{T}$$  
    {: .prompt-tip .small-math }  

  - **Algorithm**
      {% highlight python %}
      while True:
        theta_grad = evaluate_gradient(J, corpus, theta)
        theta = theta - alpha * theta_grad {% endhighlight %}
